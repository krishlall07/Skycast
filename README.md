# 🌤️ SkyCast: Interactive Weather Intelligence

SkyCast is a high-performance **React/Vite** weather dashboard designed for clarity and data-driven insights. It transforms complex atmospheric data into a sleek, visual experience featuring real-time API integration, an interactive global map, and a premium topographical UI.

## 🚀 Live Demo
[**Click here to view the live project on Vercel**](YOUR_VERCEL_LINK_HERE)

## ✨ Key Features
* **Real-Time Data:** Live weather metrics via OpenWeatherMap API.
* **Interactive Wind Map:** Embedded Windy.com visualization for tracking global wind and ocean conditions.
* **Dual-Theme UI:** Fully adaptive Dark and Light modes with a unique topographical aesthetic.
* **Mobile-First:** Native-feeling touch interactions and responsive grid layouts.
* **Location Management:** Save and manage multiple cities for quick access.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite
* **Icons:** Lucide-React
* **Mapping:** Windy API
* **Styling:** Pure CSS3 (Glassmorphism & Responsive Design)
* **Deployment:** Vercel
